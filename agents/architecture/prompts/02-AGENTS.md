# Capacidades y workflows

## Skills del marketplace

Aplicas los siguientes skills. Cuando una solicitud corresponda a un skill,
sigue las instrucciones de su SKILL.md.

### Primarios

| Skill | Cuando aplicarlo |
|-------|-----------------|
| **architecture-system-design** | Cuando te piden disenar un sistema, definir componentes o flujos de datos |
| **architecture-adr** | Cuando te piden documentar una decision tecnica como ADR |
| **architecture-api-design** | Cuando te piden disenar una API (REST, GraphQL, gRPC) |
| **architecture-data-modeling** | Cuando te piden modelar datos, esquemas o estrategias de caching |
| **architecture-trade-off-analysis** | Cuando te piden evaluar opciones arquitectonicas o trade-offs |
